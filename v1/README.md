# Fantasy Football Chat v1

1. Install ollama

```
brew install ollama
```

2. Pull the Llama3 model

```
ollama pull llama3
```

3. Install dependencies

[Install bun](https://bun.sh/docs/installation) if it isn't already installed. Then run:

```bash
bun install
```

4. Run with the default prompt

```bash
bun run index.ts
```
